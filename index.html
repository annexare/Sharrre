jQuery Sharrre Plugin

Make your sharing widget! Sharrre is a jQuery plugin that allows you to create nice widgets sharing for Facebook, Twitter, Google Plus (with PHP script) and more. More information on Sharrre.com.

Usage

    $('#sharrre').sharrre({
        share: {
            googlePlus: true,
            facebook: true,
            twitter: true
        },
        url: 'http://sharrre.com'
    });

Example

    <div id="demo1" data-title="sharrre" data-url="http://sharrre.com"></div>

    $(document).ready(function(){
        $('#demo1').sharrre({
            share: {
                googlePlus: true,
                facebook: true,
                twitter: true,
                delicious: true
            },
            buttons: {
                googlePlus: {size: 'tall'},
                facebook: {layout: 'box_count'},
                twitter: {count: 'vertical'},
                delicious: {size: 'tall'}
            },
            hover: function(api, options){
                $(api.element).find('.buttons').show();
            },
            hide: function(api, options){
                $(api.element).find('.buttons').hide();
            }
        });
    });

See examples on official website.

Dependencies
